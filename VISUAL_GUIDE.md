## 🎨 Color Coding System

### Risk/Severity Indicators
- 🟢 **Green (Low/Mild)**: Score 0-30
  - Safe range
  - Preventive measures sufficient
  - Continue monitoring

- 🟡 **Yellow (Moderate)**: Score 31-60
  - Action recommended
  - Lifestyle changes needed
  - Consider professional consultation

- 🔴 **Red (High/Severe)**: Score 61-100
  - Immediate attention required
  - Professional evaluation recommended
  - Urgent interventions needed

### Priority Levels
- 🔴 **URGENT**: Immediate action required (red background)
- 🟠 **HIGH**: Important, act within days (orange background)
- 🟢 **RECOMMENDED**: Beneficial, act when convenient (green background)

## 🎯 Key Visual Elements

### 1. Icons Used
- 🧠 Brain: AI analysis, intelligence
- 👁️ Eye: Vision, eye health
- 🔴 Red Circle: Symptoms, warnings
- 🟡 Yellow Circle: Risk factors, caution
- 🟢 Green Circle: Prevention, healthy
- 📚 Books: Knowledge, library
- 🔍 Magnifying Glass: Search, exploration
- ⚠️ Warning Triangle: Alerts, attention needed
- ✅ Check: Validated, good state
- 📊 Chart: Metrics, data
- 💡 Lightbulb: Insights, tips
- 🎯 Target: Goals, focus

### 3. Button Styles
- **Primary**: Blue background, white text (main actions)
- **Secondary**: Gray background, dark text (alternative actions)
- **Link**: Transparent, colored text (navigation)
- **Urgent**: Red background, white text (critical actions)
