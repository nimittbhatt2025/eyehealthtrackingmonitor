## 💡 Key Insights from This Session

1. **Voice Recognition is Game-Changer**
   - Solves UX problem: users at 40cm can't reach screen
   - Enables true clinical-style testing
   - Browser support good (Chrome/Safari, not Firefox)

2. **Eye Coverage Verification is Critical**
   - Monocular tests (Visual Acuity, Amsler Grid) need this
   - Brightness-based detection works well
   - Webcam access requires HTTPS or localhost

3. **Glasses Check is Mandatory**
   - Essential for accurate baseline measurements
   - Two-step flow: "Do you wear?" → "Have you removed?"
   - Educational content helps users understand why

4. **Emoji Removal Takes Time**
   - 100+ emoji instances across 15+ files
   - Each replacement needs SVG icon or text alternative
   - Systematic approach necessary (grep search → patches → apply)

5. **Integration is Iterative**
   - Start with one test (Visual Acuity) ✅
   - Apply lessons learned to next tests
   - Document patterns for consistency
